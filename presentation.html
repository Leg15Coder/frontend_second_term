    Какую проблему решаем

    Понимаем боль. У нас есть решение.

    Привычки не приживаются

    Люди бросают привычки через несколько дней. Нет визуального прогресса, нет ощущения достижения, мотивация угасает.

    Цели остаются мечтами

    Цели расплывчатые и не разбиты на конкретные шаги. Без структуры и дедлайнов они так и остаются в списке желаний.

    Скучные трекеры убивают мотивацию

    Существующие приложения сухие и функциональные. Нет эмоций, визуального кайфа, ощущения победы над собой.

    Краткий обзор API

    Основные ресурсы приложения:

    • /api/habits — привычки (GET/POST), PATCH/DELETE по id.
    • /api/goals — цели (GET/POST) и управление ими.
    • /api/me — профиль текущего пользователя (GET/PATCH).

    Для подробной документации откройте полную страницу API:

    Полная документация APIОткрыть Swagger UI

    Для разработчиков

    Кратко об архитектуре и инструментах, которые используются в проекте:

    • Frontend: React + TypeScript, Vite, Tailwind CSS.
    • State: Redux Toolkit (slices, createAsyncThunk) + React Router.
    • Backend: Firebase (Firestore) / REST API.
    • Tests & CI: Vitest (unit), Storybook + visual tests, Cypress (E2E), GitHub Actions для автоматизации.
